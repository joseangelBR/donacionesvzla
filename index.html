¿Cómo funciona?

Instalas un pequeño código una sola vez. El listado de organizaciones vive en la nube (un archivo config.json) y se actualiza para todos a la vez.

1. Elige el formato

Plugin de WordPress, código para Wix, iframe o un simple <script> para cualquier web.

2. Pega el código

Aparece una franja discreta abajo (o un botón / popup, tú eliges). No bloquea tu contenido.

3. Listo y siempre al día

Cuando se verifican nuevos canales de ayuda, tu widget se actualiza solo. No tienes que tocar nada.

Genera tu código

Personaliza y copia. Funciona en cualquier sitio HTML, y en Wix con bloque de código.

Vacío = mostrar todas y mantenerse actualizado. Si marcas algunas, solo esas se mostrarán (los enlaces siguen actualizándose y las que se retiren por seguridad desaparecen).

Instalar en WordPress

Con panel de ajustes, bloque de Gutenberg y shortcode.

⬇ Descargar plugin (.zip)

  1. Descarga el .zip del plugin.
  2. En tu WordPress: Plugins → Añadir nuevo → Subir plugin, elige el zip y actívalo.
  3. Ve a Ajustes → Ayuda Venezuela y elige modo, idioma y color.
  4. Opcional: coloca el bloque “Ayuda Venezuela” o el shortcode [donaciones_vzla] dentro de una entrada o página.

El plugin no incluye el listado: lo lee del config.json remoto. Actualizar las organizaciones nunca requiere actualizar el plugin.

Instalar en Wix

  1. Edita tu sitio → Añadir (+) → Insertar → Insertar HTML (Embed HTML).
  2. Elige “Código” y pega el Script universal del generador de arriba. Si Wix bloquea scripts en tu plan, usa la opción iframe.
  3. Publica el sitio.
