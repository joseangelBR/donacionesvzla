Muestra una franja, botón o popup con canales verificados para donar a Venezuela. El listado se actualiza solo desde la nube.

== Description ==

Añade a tu sitio un widget discreto con formas verificadas de ayudar a Venezuela tras el terremoto.

* **No recibe dinero**: cada botón enlaza directo a la página oficial de la organización.
* **Se actualiza solo**: el listado vive en un archivo remoto (config.json). Cuando se verifican nuevos canales, tu sitio se actualiza sin que reinstales el plugin.
* **Tres formas de mostrarlo**: franja fija abajo (recomendado), botón flotante lateral o popup al entrar.
* **Ligero y accesible**: se aísla del CSS de tu tema (Shadow DOM), es responsive y accesible por teclado.
* **Bloque y shortcode**: inserta el listado dentro de una entrada con el bloque “Ayuda Venezuela” o `[donaciones_vzla]`.

Transparencia: proyecto agregador independiente, sin afiliación con las organizaciones y sin comisión.

== Installation ==

1. Sube el `.zip` en Plugins → Añadir nuevo → Subir plugin, y actívalo.
2. Ve a Ajustes → Ayuda Venezuela y elige modo, idioma y color.
3. (Opcional) Inserta el bloque “Ayuda Venezuela” o el shortcode `[donaciones_vzla]` en una página.

== Frequently Asked Questions ==

= ¿El plugin gestiona pagos? =
No. Solo redirige a las páginas oficiales de donación de cada organización.

= ¿Tengo que actualizar el plugin cuando cambie el listado? =
No. El listado se carga en vivo desde config.json. Solo actualizarás el plugin para nuevas funciones.
